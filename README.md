## üöÄ Deployment (The Hydra Protocol)

This project is optimized for **Vercel Serverless Functions**. This ensures high availability, auto-scaling, and dynamic IP rotation.

### 1-Click Deploy
1. Fork this repository.
2. Go to [Vercel Dashboard](https://vercel.com).
3. Import your forked repo.
4. Framework Preset: **FastAPI** (or Other).
5. **Deploy.**

### Manual Setup
```bash
# Clone the nexus
git clone https://github.com/david-aistudio/KagentaAPI.git
cd KagentaAPI

# Activate virtual environment
python3 -m venv venv
source venv/bin/activate

# Install neural pathways
pip install -r requirements.txt

# Ignite the core
python3 main.py
```
